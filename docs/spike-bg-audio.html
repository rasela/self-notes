Background-Audio Spike

Tests whether iOS Safari keeps feeding gapless audio while the screen is locked. The load-bearing question for beats.html streaming.

How to run the test: wired or BT headphones, plug in or charged. Tap Start, confirm you hear a steady tone, then lock the phone and pocket it. Wait ~3–5 min, unlock, and read the log + verdict below. The tone uses a continuous phase accumulator across chunks, so any gap you hear is a swap stall, and any pitch jump would be a phase break (there should be none). Each chunk swap is logged with wall-clock drift, so even if you didn't hear it, the log shows whether JS kept up in the background.
